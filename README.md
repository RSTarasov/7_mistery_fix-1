# Quadratic Equations Solver

If you have an equation of the form "ax2 + bx + c = 0", this function can solve it for you.
Just enter the values of a, b and c as parameters.

# How to Use

```
get_roots(a, b, c)
```

# How to Launch Tests

```bash
python tests.py
```
